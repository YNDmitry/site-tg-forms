## Installation

```bash
$ git clone https://github.com/YNDmitry/site-tg-forms

$ cd site-tg-forms

$ npm install

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```
